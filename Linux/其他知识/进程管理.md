## ps

```bash
### 特有
ps l
# F			(0:user 4:root)
# PPID 	  	父进程
# NI PRI 	nice,priority值
ps u
# USER
# START 	产生进程的时间
# %CPU %MEM 占用率

### 共有
# PID
# VSZ  		虚拟内存占用
# RSS  		物理内存占用
# STAT 		进程状态
	# R(运行) 	 D/S(不可中断/可中断 睡眠) T(停止) 	Z(僵死) 
	# s(包含子进程) l  (多线程)				+(位于后台)	 L(锁入内存)
	# <(高优先级)   N  (低优先级)
# COMMAND   进程名称
# TTY		产生进程的终端
```

#### 查看全部进程的信息

```bash
ps -aux
ps -le
```
